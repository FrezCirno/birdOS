# OS
> 同济大学操作系统课程项目
>
> 利用C语言，汇编实现的简易操作系统    

## 实现功能

- 系统级
  - 重新实现引导扇区
  - 多级文件目录
  - 图形化系统
  - 进程管理
  - 多级队列调度


- 用户级
  
- 游戏：2048、扫雷、贪吃蛇、推箱子、计时器、计算器、日历、五子棋等
